# T90 Titans League Season 5 Maps

Map pack for the [T90 Titans League Season 5](https://liquipedia.net/ageofempires/T90_Titans_League/5/Platinum_League) tournament, hosted by [T90Official](https://www.twitch.tv/t90official).
Install the mod by searching for "T90 Titans League - Season 5 - Maps" in the in-game mod center, or by using the following link:

[https://www.ageofempires.com/mods/details/445905/](https://www.ageofempires.com/mods/details/445905/)

The map pool includes the following 15 maps:

- [African River](https://liquipedia.net/ageofempires/African_River) by cebdos
- [Bypass](https://liquipedia.net/ageofempires/Bypass) by Chrazini
- [Coast Arena](https://liquipedia.net/ageofempires/Coastarena) by TheMadCADer
- [Dzimbabwe](https://liquipedia.net/ageofempires/Dzimbabwe) by Marintho
- [High Tides](https://liquipedia.net/ageofempires/High_Tides) by AlgernonR
- [Monocle](https://liquipedia.net/ageofempires/Monocle) by Chrazini
- [Mudflow](https://liquipedia.net/ageofempires/Mudflow) by Chrazini
- [Northern Crossings](https://liquipedia.net/ageofempires/Northern_Isles) by Chrazini
- [Quarry](https://liquipedia.net/ageofempires/Quarry) by HenkDeSuperNerd
- [Reverse Acropolis](https://liquipedia.net/ageofempires/Reverse_Acropolis) by YoungPanda95
- [Seychelles](https://liquipedia.net/ageofempires/Seychelles) by Deathcounter
- [Slopes](https://liquipedia.net/ageofempires/Slopes) by AlgernonR
- [Stranded](https://liquipedia.net/ageofempires/Stranded) by Chrazini
- [Tulou](https://liquipedia.net/ageofempires/Tulou) (Regicide) by Martintho
- [Valley](https://liquipedia.net/ageofempires/Valley) by Forgotten Empires

## Build Process

These files are the source files, not the final map scripts included in the mod.
The maps require [T-West's preprocessor](https://github.com/twestura/RMS-Preprocessor) in order to be built.
